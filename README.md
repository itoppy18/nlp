### dependencyAnalyzer(text)
textを係り受け解析します。
\n「新しいご馳走の発見は、新しい星の発見よりも人々を幸せにする。」

```python3:
[
["新しい", "ご馳走の"], 
["ご馳走の", "発見は、"], 
["発見は、", "幸せにする。"], 
["新しい", "星の"], 
["星の", "発見よりも"], 
["発見よりも", "幸せにする。"], 
["人々を", "幸せにする。"]
]
```
### inflections(word)
wordの活用を取得します。動詞と形容詞に対応しています。
\n「輝く」

```python3:
[["輝か", "輝こ"], ["輝き", "輝い"], "輝く", "輝く", "輝け", "輝け"]
```
### auxiliaryVerb(word, mode)
wordの助動詞活用を取得します。
\n「輝く」 mode="hope"

```python3:
"輝きたい"
```
