### inflections(word)
wordの活用を取得します。動詞と形容詞に対応しています。 

```python3:
[["輝か", "輝こ"], ["輝き", "輝い"], "輝く", "輝く", "輝け", "輝け"]
```
### auxiliaryVerb(word, mode)
wordの助動詞活用を取得します。  「輝く」 mode="hope"

```python3:
"輝きたい"
```
